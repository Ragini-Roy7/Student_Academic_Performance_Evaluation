
# 🎓 Student Academic Performance Evaluation

## 📌 Project Overview
This project focuses on analyzing and evaluating **student academic performance** using Python-based data analysis and visualization techniques.  
It applies basic **Exploratory Data Analysis (EDA)** to identify patterns, trends, and relationships within academic data.

The project is designed for **learning purposes**, especially for students exploring **data analysis, statistics, and machine learning foundations**.

---

## 🧰 Technologies Used
- **Python**
- **NumPy** – numerical computations
- **Pandas** – data manipulation and analysis
- **Matplotlib** – data visualization
- **Scikit-learn** – statistical analysis (correlation, preprocessing)

---

Student_Academic_Performance_Evaluation/

---

## 📊 Key Analysis Performed
- Data loading and inspection
- Handling missing and inconsistent values
- Statistical summary of features
- **Pearson correlation analysis**
- Data visualization using:
  - Histograms
  - Bar charts
  - Heatmaps
- Performance trend analysis

---

## 📈 Output & Insights
- Identifies relationships between academic factors
- Helps understand how different attributes affect student performance
- Visual representation for better interpretation of data
